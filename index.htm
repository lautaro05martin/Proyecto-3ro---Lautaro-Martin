Proyecto Final de Carrera

MOOD - Sistema de Gestión para Gimnasios

MOOD es un sistema de gestión diseñado para gimnasios que permite digitalizar los procesos administrativos, reemplazando el uso de planillas manuales. El sistema facilita el control de ingresos, egresos, gestión de abonos y administración del personal, mejorando la organización y reduciendo errores.

  • Nombre: Gimnasio MOOD
  • Rubro: Actividad física y entrenamiento
  • Responsable:
  • Email:

Definición del negocio

El gimnasio MOOD ofrece servicios de entrenamiento físico, clases grupales y planes personalizados para sus clientes. Actualmente gestiona sus operaciones administrativas de forma manual.

Alcance del sistema

El sistema propuesto será una aplicación administrativa que permitirá controlar ingresos, egresos, abonos, sueldos y estadísticas del negocio.

Problema a resolver

Actualmente, la gestión del gimnasio se realiza de forma manual, lo que genera errores en el control de ingresos, dificultad para llevar registros claros y pérdida de información.

Propuesta de valor

El sistema MOOD permitirá automatizar la gestión del gimnasio, reduciendo errores humanos, mejorando el control financiero y optimizando la administración del personal. Esto facilitará la toma de decisiones y aumentará la eficiencia del negocio.

Funcionalidades principales

  • Registro de cierre de caja diario
  • Control de ingresos y egresos
  • Gestión de abonos (pagos realizados y pendientes)
  • Control de horarios del personal
  • Gestión de sueldos (solo visible para administradores)
  • Estadísticas mensuales del negocio

Roles del sistema

  • Administrador: acceso total al sistema
  • Recepcionista: gestión de caja y abonos
